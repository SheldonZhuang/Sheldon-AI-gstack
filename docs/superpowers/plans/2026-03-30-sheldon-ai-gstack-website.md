# Sheldon AI gstack 网站实现计划

**日期：** 2026-03-30
**规格文档：** `docs/superpowers/specs/2026-03-30-sheldon-ai-gstack-website-design.md`
**目标：** 构建苹果风格的中英双语静态网站，展示 gstack 的 20 个 AI 角色

---

## 实现策略

**技术栈：** 纯静态（HTML + CSS + JS），零构建工具，零依赖
**部署：** GitHub → Vercel 自动 CI/CD
**开发顺序：** 骨架 → 样式 → 数据 → 交互 → 验证

---

## 风险与依赖

**风险 1：20 个角色数据量大**
- 缓解：分 5 批次填充，每批次验证一次

**风险 2：纯静态无构建工具，调试依赖浏览器**
- 缓解：使用 Live Server 或 Python `http.server` 本地预览

**风险 3：中文翻译质量**
- 缓解：基于 gstack SKILL.md 原文，保持技术准确性

**依赖：**
- gstack 仓库可访问（https://github.com/garrytan/gstack）
- Vercel 账号已准备
